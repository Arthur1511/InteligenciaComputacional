# InteligenciaComputacional
Trabalho desenvolvido para a disciplina de Inteligencia Computacional para Otimização no semestre 2018.2

Esse trabalho consiste na implementação de métodos construtivos para gerar uma solução inicial, heurísticas e 
meta-heurísticas de refinamento de soluções tendo como base o TSP (Traveler Salesmen Problem).

Work developed for the discipline of Computational Intelligence for Optimization in the semester 2018.2

This work consists of the implementation of constructive methods to generate an initial solution, heuristics and 
metaheuristics based on TSP (Traveler Salesmen Problem).
